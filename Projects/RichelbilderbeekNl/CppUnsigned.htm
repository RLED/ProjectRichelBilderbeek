(C++) unsigned

 

unsigned is a keyword that modifies a data type to hold only positive values.

 

Avoid using unsigned in a class interface; use int instead [1]. Consider avoiding unsigned [2] or unsigned arithmetic [3] as using an unsigned instead of an int to gain one more bit to represent positive integers is almost never a good idea [4]. If you do use unsigned, always check your ranges [6] (note that [6] is in favor of using unsigned). View signed to unsigned and unsigned to signed conversions with suspicion [5,7].

 

int main()
{
  const unsigned int x = 123; //x can only contain positive integers
}
