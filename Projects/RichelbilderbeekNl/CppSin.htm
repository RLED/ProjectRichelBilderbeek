( C++ ) std::sin

 

Standard function for calculating the sine of an angle, where the angle is in radians.

 

To use std::sin , the STL header file cmath.h must be #include d.

 

 

#include <cmath>

#include <cassert>

 

int main ()

{

const double x = 0.5 * M_PI;

const double y = std::sin(x);

assert(y == 1.0);

}
