(C++) Multithreaded RTL must be selected

 

Compile error.

 

 

 

 

 

Full error message

 

 

[C++ Fatal Error] push_options.hpp(58): F1003 Error directive: Multithreaded RTL must be selected.

 

 

 

 

 

 

Cause

 

IDE: C++ Builder 6.0

Compiler: Borland BCC32.EXE version 6.0.10.157

Boost version: 1.35.0.

 

 

#include <boost/asio.hpp>

 

 

 

 

 

 

Solution

 

You must turn on multi-thread applications.

 

0.               'File | Close All'

1.               'File | New | Other'

2.               Select 'Console Wizard'

3.               Also check 'Multi-threaded'
