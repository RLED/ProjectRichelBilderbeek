(Tool) OpenFOAM

 

OpenFOAM is 'a C++ toolbox for the development of customized numerical solvers, and pre-/post-processing utilities for the solution of continuum mechanics problems, including computational fluid dynamics (CFD)' [1]

 

This page contains my notes about OpenFOAM. OpenFOAM is originally developed under Linux, my need is to get it working under Windows.

I downloaded a Windows installer of OpenFOAM version 1.5 from the Sourceforge project 'openfoam-mswin', which worked conveniently.

OpenFOAM collaborates with other tools:

 

  • OpenFOAM designer (main screen screenshot (png), about screen screenshot (png)) (executable at C:\cfd\OpenFOAM-1.5\bin\ofdesign.exe)
  • ParaView (screenshot (png) (executable at "C:\Program Files\ParaView 4.0.1\bin\paraview.exe")

 

 

 

 

 

OpenFOAM designer: not a valid case

 

In OpenFOAM designer, I do 'File | Open case', then paste the following two paths:

  • C:\cfd\OpenFOAM-1.5\templates\icoFoam\cavity
  • D:\Airbearing\cpp\build-MeshCreator-Desktop_Qt_5_1_1_MinGW_32bit-Debug

The second path results in this error:

The folder structure of the working case is as follows:

  • (no files in case folder itself)
  • 0/p
  • 0/U
  • constant/transportProperties
  • constant/polyMesh/blockMeshDict
  • system/controlDict
  • system/fvSchemes
  • system/fvSolution

 

The folder structure of the defective case is as follows:

  • (no files in case folder itself)
  • 0 (empty folder)
  • constant/polyMesh/boundary
  • constant/polyMesh/faces
  • constant/polyMesh/neighbour
  • constant/polyMesh/nodes
  • constant/polyMesh/owner
  • system (empty folder)

 

Copying the following file made the case working:

  • system/controlDict

    • Not necessarily needed are:

    • 0/p
    • 0/U
    • constant/transportProperties
    • system/fvSchemes
    • system/fvSolution
