(Tool) DotMatrix

 

DotMatrix is a tool to display text as dot-matrix characters (5 (width) x 7 (height) pixels, without spacer pixels), like on your microwave display.

 

The program relies heavily on the function DotMatrix. It is the function that draws a character on a TImage. The advantage to VCL functions like TextOut is that you can be sure about the font's dimensions and character placement.
