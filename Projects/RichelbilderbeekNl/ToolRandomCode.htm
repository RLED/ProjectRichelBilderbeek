(Tool) Random Code

 

A simple tool that creates random C++ code. I wrote it for the forums I visited: when somebody was begging for code ('Coult u pleaz gimme code 4 peer2peer programm?'), I posted a piece of random generated code with the comment 'Here it is, it just needs to have some bugs fixed!'. Programmers' humor is great, isn't it?

 

Random Code can be used on my interactive website.
