(Tool) SolvePuzzleX

 

SolvePuzzleX is a tool to solve the puzzle given to me by X: given the set of numbers of {1,3,4 and 6}, can you make the value of 24 by only using multiplication, division, subtraction and addition (no square roots, no exponents, no geometry) use each number exactly once, and without 'glueing' two numbers apart?
