(Tool) SecretMessage

 

SecretMessage is a tool to add a secret message to a 24-bit bitmap. These secret messages can also be extracted again.

SecretMessage information

 

 

 

 

 

SecretMessage version 1.0 operating system(s) or programming environment(s)

 

  • Windows XP

IDE(s):

  • C++ Builder 6.0 Enterprise edition

Project type:

  • GUI application

Compiler(s):

  • BCC32.EXE supplied with C++ Builder 6.0 Enterprise edition

Libraries used:

  • Boost: version 1.40
