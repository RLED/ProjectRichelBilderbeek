(Tool) ThresholdFilterer

 

ThresholdFilterer is a tool to perform threshold filter operations and demonstrates the function ThresholdFilter.

 

ThresholdFilterer is programmed in C++ using the IDE C++ Builder 6.0. It uses the STL, VCL and Boost libraries.
