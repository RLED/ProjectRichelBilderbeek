(C++) Test/test

 

Test/test has multiple meanings:

  • Testing: the process of validating your code
  • Boost.Test: a Boost library

 

 

 

 

 

Testing

 

Testing is the process of validating your code.

 

Distributing testing throughout the design hierarchy can be much more effective per testing dollar than testing at only the highest-level interface' [1]. Testing a component in isolation is an effective way to ensure reliability' [2]. Thorough regression testing is expensive but essential; the appropriate time to create thorough regression tests is tied to the stability of the subsystem to be tested' [3].
