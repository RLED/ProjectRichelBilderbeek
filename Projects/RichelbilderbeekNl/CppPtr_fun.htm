(C++) std::ptr_fun

 

An adapter to be able to use the algorithm for_each on a function of T stored in a container as T, instead of using loops. Prefer algorithm calls over hand-written loops [1,2].

 

 

 

 

 

Example

 

#include <algorithm>

#include <cctype>

#include <string>

 

//From http://www.richelbilderbeek.nl/CppStrToUpper.htm

const std::string StrToUpper (std::string s)

{

std::transform(s.begin(), s.end(), s.begin(),std::ptr_fun(std::toupper));

return s;

}
