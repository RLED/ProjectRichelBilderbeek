(Tool) ndsmake

 

ndsmake is a tool to generate an NDS makefile from a Qt Creator project file.

ndsmake wishlist

  • Support multi-unit projects
  • Support multi-folder projects
  • Support graphics compiling (with devkitPro's grit)
