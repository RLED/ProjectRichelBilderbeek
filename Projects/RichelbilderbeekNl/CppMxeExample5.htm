(C++) MXE example 5: Hello Boost.Regex

 

MXE example 5: Hello Boost.Regex is an MXE example to cross-compile a Hello Boost program from GNU/Linux to Windows.

 

Note that for me, this code does not cross-compile.

 

  • Download the Qt Creator project 'CppMxeExample5' (zip)

 

 

 

 

 

Technical facts

 

Application type(s)

  • Desktop application

Operating system(s) or programming environment(s)

  • Lubuntu 12.04 (precise)
  • Ubuntu 12.04 (precise)

IDE(s):

  • Qt Creator 2.4.1

Project type:

  • Console application

C++ standard:

  • C++98

Compiler(s):

  • G++ 4.6.3

Libraries used:

  • Boost: version 1.46.1
  • STL: GNU ISO C++ Library, version 4.6.3

 

 

 

 

 

Qt project file: CppMxeExample5.pro

 

QT       -= core
QT       -= gui
TARGET = CppMxeExample5
LIBS += -lboost_regex
CONFIG   += console
CONFIG   -= app_bundle
TEMPLATE = app
SOURCES += main.cpp

 

 

 

 

 

main.cpp

 

#include <iostream>
#include <boost/regex.hpp>

int main(int argc, char* argv[])
{
  boost::regex r("Hello Boost.Regex");
  std::cout << r.str() << '\n';
}

 

 

 

 

 

crosscompiletowindows.sh

 

#!/bin/sh
#From http://richelbilderbeek.nl/CppMxe.htm

echo "Cross compiling to Windows"

echo "1/2: Creating Windows makefile"
i686-pc-mingw32-qmake CppMxeExample5.pro

echo "2/2: making makefile"

make

echo "Done"
