FileToTStringList

 

Use TStringList's method LoadFromFile.

 

 

TStringList * const myStringList = new TStringList;

myStringList->LoadFromFile("fileName");

delete myStringList;

 

 

Using std::auto_ptr:

 

 

#include <memory>

 

std::auto_ptr<TStringList> myStringList(new TStringList);

myStringList->LoadFromFile("fileName");

//delete takes place automatically
