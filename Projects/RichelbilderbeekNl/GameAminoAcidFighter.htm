(C++) GameAminoAcidFighter

 

Amino Acid Fighter started as a Quickbasic game in 2000. Later I ported it to C++, but have not yet finished it and lost the code as well!

 

Amino Acid Fighter is a game in which amino acids do battle to a maximum of four players.

 

Amino Acid Fighter is programmed in C++ using the IDE C++ Builder 6.0. It uses the VCL and Boost libraries.
